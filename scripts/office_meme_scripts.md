## POSTING NOTES

- **Caption style (TikTok/IG):** Keep it blank or one line — the video IS the caption
- **Hashtags:** #productmanagement #softwareengineering #aitools #techmemes #startuplife #pm #buildinginstartups
- **Best time to post:** Tue–Thu, 9–11am or 6–8pm CST
- **Don't explain the joke.** Ever.

## RE-RENDER
If you want to change any caption text, edit `scripts/render_reels.py` and run:
```bash
python3 scripts/render_reels.py
```
Individual clips can be re-rendered by commenting out the others in the CLIPS list.
